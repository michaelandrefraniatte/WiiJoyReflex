Using Joycon Left and Wiimote as Keyboard and Mouse in FPS Games

The program allow to use Wiimote and Joycon left in FPS games with easy Configurations. It change in the game as soon as you change Configurations in the program. It's easy to understand and easy to win with this program. You can play all your FPS games the way you want if the Keyboard and Mouse is well supported in. Let's see...

Considerations About the Program

The Keyboard key F9 enable/disable the controls. After the program start, you shall put Joycon left and Wiimote in pairing mode, by pressing the buttons to connect it via Bluetooth. The connection is automatic with just that, and the disconnection is totally automatic after you close the program.
